# net-bottleneck

OS级别网络瓶颈分析Skill，分析网络带宽、延迟、连接状态、协议栈效率。

## 总体功能

分析OS级别网络性能瓶颈：

1. **环境检查** - 连接状态、Socket统计
2. **连接分析** - TCP连接状态分布
3. **协议分析** - 重传、超时、窗口
4. **延迟测试** - 网络延迟测量

**Scope**: 仅分析OS网络栈，不包含应用网络

## 前置依赖

### 必需依赖

- **remote-execution skill** - SSH远程连接

### 工具依赖

- `ss`, `netstat`
- `ping`

## 用法

### 自然语言输入示例

```
分析192.168.1.100的网络瓶颈
```

```
网络延迟高，帮我看看是哪里的问题
```
