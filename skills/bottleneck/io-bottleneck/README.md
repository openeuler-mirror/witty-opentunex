# io-bottleneck

OS级别IO瓶颈分析Skill，分析磁盘利用率、IO等待、队列深度等。

## 总体功能

分析OS级别IO性能瓶颈：

1. **环境检查** - 磁盘类型、调度器、内核支持
2. **指标收集** - iostat/vmstat/mpstat
3. **瓶颈识别** - 识别IO饱和、延迟高、队列积压
4. **优化建议** - IO层面优化策略

**Scope**: 仅分析OS级别，不包含应用层IO

## 前置依赖

### 必需依赖

- **remote-execution skill** - SSH远程连接

### 工具依赖

- `iostat` (sysstat)
- `vmstat`, `mpstat` (sysstat)
- `pidstat` (sysstat)

## 用法

### 自然语言输入示例

```
分析192.168.1.100的IO瓶颈
```

```
磁盘IO高，帮我看看是哪里问题
```
